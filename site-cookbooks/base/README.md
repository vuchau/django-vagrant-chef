Base cookbook
======================

Installs/Configure basic Debian -  Ubuntu box for development

Requirements
------------

### Platform:

The following platform families are supported:

* Debian - Ubuntu

### Cookbooks:

* packages


Atrributes:
-----------

* `['base']['packages']` - a list of default packages to be installed
